# React calendar

Project developed to apply studies about React. Using:
- React
- Recoil
- Axios to requests from api
- ESLint
- Css modules
- Sass

<img src="./public/app.png" width="60%">

## Usage

- Install
> npm install

- Init api
> json-server --watch db.json --port 3004

- Init app
> npm start
